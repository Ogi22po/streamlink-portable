Direct usage:
Run %STREAMLINK_EXEC_FILE% and type the desired arguments

Usage from cmd:
%STREAMLINK_EXEC_FILE% ARGUMENTS

Notes:
-To disable CMD hints create a file named 'NO_CMD_HINTS'
